# LIVE EFFECTS  (PageItem.applyEffect)
# ============================================================
#
# applyEffect(liveEffectXML: string) takes a Live Effect XML string.
# The exact effect name strings are version-specific and NOT documented
# in a public structured source. Do not guess them.
#
# To discover valid effect names for the running Illustrator installation:
#   var names = app.liveEffectsList;  // returns Array of effect name strings
#
# To read back the XML of an effect already applied to an item:
#   var xml = item.effectsData;       // returns XML string of all applied effects
#
# Workflow for applying an effect by name:
#   1. Call app.liveEffectsList to get the authoritative list for this install
#   2. Match the user's intent to a name from that list
#   3. Build the XML string and call item.applyEffect(xml)
#
# Do not hardcode effect name strings — they vary by Illustrator version and locale.
